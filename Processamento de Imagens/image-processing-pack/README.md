# image-processing

Description. 
The package package_imageTest is used to:
	Processing:
		- Histogram Matching
		- Structural Similarity
    	- Resize images
	Utils:
		- Read image
		- Save image
		- Plot image
		- Plot Result
		- Plot hystogram



## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install image-processing

```bash
pip install image-processing
```
